  **专家知识库**： 对海量内容加以整理，使成为大型的专家知识库模型。
  
  **手机摄像机**：调用手机摄像头，获取高精度图片
  
 **热点提示**：比对图片，检索相关的知识点，提供对用户有帮助的“热点知识”。
